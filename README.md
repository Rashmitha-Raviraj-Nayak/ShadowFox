# ShadowFox
Repository contains tasks, projects, and practice work completed during the ShadowFox Java Development Internship. It covers core Java, OOP principles, collections, exception handling, file handling, JDBC, multithreading, and mini-projects that demonstrate structured coding, clean design, and real-world problem-solving skills in Java development.
